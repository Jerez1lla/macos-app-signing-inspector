# Changelog

## [1.0.0] - 2026-07-14

### Added

* Native macOS Inspector for application metadata, code signatures, Designated Requirements, Gatekeeper status, notarization information, and main-executable architectures.
* Copy actions and diagnostics for important inspection values.
* Independent graphical Policy Builder workspace with specific-application Allow and Deny rules.
* Developer Team ID, documented `*APPLE*`, Jamf convenience, `PathPrefix`, and `AlwaysAllowManagedApps` rule support.
* Duplicate, validation, redundancy, broad-rule, and allow-only policy warnings.
* Complete `com.apple.configuration.app.settings` JSON preview, copy, and local export.
* Native sidebar navigation, keyboard commands, accessibility refinements, About information, and application icon resources.
* Deterministic unit coverage for parsing, inspection state, architecture mapping, policy validation, navigation, and JSON generation.
