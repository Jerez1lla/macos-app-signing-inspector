# Support

Use GitHub Issues for reproducible bugs and focused feature proposals.

For bugs, include the App Signing Inspector version, macOS version, reproduction steps, expected result, actual result, and sanitized diagnostics. macOS beta behavior may change between releases.

Do not upload proprietary applications, credentials, private keys, signing certificates, internal URLs, sensitive Team IDs, or confidential diagnostics. Security vulnerabilities must follow [SECURITY.md](SECURITY.md) and must not be disclosed in a public issue.
